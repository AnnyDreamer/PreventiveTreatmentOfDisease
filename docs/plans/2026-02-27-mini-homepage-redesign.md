# 小程序首页重设计方案

**日期**: 2026-02-27
**范围**: `apps/mini/src/pages/index/`
**状态**: 已确认，待实施

---

## 问题描述

当前首页存在以下问题：
- 布局散落拥挤，各区块视觉权重接近，缺乏焦点
- 打卡提醒条 + 节气卡分占两个独立区块，空间浪费
- 2×2 服务宫格竖排占用纵向空间过多
- 中医水墨风偏传统，用户群希望更现代化的健康 App 感

---

## 设计目标

1. **清晰的信息层级**：体质状态 → 今日提醒 → 功能入口 → 养生内容
2. **清新绿色健康风**：替换棕黄水墨风为清新绿白系
3. **节省纵向空间**：打卡 + 节气合并为单行横条
4. **功能入口横排化**：核心服务由 2×2 改为 1×4 横排

## 页面结构

### 1. 顶栏 Banner（简洁版）
- 背景：浅绿渐变 `#e8f7ef → #f4fbf7`
- 左侧：问候语（小字）+ 昵称（大字）
- 右侧：头像，边框改为绿色 `rgba(45,170,111,0.2)`
- **移除**独立节气胶囊 pill，节气信息下移合并

### 2. 体质状态主卡（核心区）
- 圆角：`20px`（比现在更饱满）
- **已评估**：绿色渐变背景 `#2daa6f → #1e8a55`，白色文字，体质名+分数+特征标签
- **未评估**：浅绿边框虚线卡，绿色圆形"辨"图标，引导文案更积极
- 点击跳转逻辑不变

### 3. 节气 + 打卡 合并横条（单行）
```
[🌿 冬 · 冬养肾，宜温补固本]    [今日打卡 ›]
```
- 高度约 60px（原来两个区块合计 >160px）
- 左侧：节气图标 + 文字（绿色调）
- 右侧：打卡入口按钮（暖黄色 `#f9a825`）
- 背景：白色卡片，左侧绿色 border-left

### 4. 核心服务（横排 1×4）
- 布局从 `flex-wrap: wrap` 的 2×2 改为横向一行不换行
- 每项：绿色圆形图标（60px）+ 标题 + 描述，居中对齐
- 去掉卡片背景，直接排列在区块内，减少视觉重量

### 5. 养生天地（2列小卡）
- 结构保持不变
- 图标主题色统一为绿色系（替换 herb/amber/purple 主题）
- 减少内边距，更紧凑
